在创作过程中，可以使用以下快捷键，让你的操作更便捷。

## 工具
|              | **MacOS**                 | **Windows**          |
| ------------ | ------------------------- | -------------------- |
| 选择工具     | `V`                       | `V`                  |
| 等比缩放工具 | `K`                       | `K`                  |
| 矩形         | `R`                       | `R`                  |
| 圆           | `O`                       | `O`                  |
| 直线         | `L`                       | `L`                  |
| 图片         | `⇧ Shift` `⌘ Command` `K` | `⇧ Shift` `Ctrl` `K` |
| 钢笔工具     | `P`                       | `P`                  |
| 铅笔工具     | `⇧ Shift` `P`             | `⇧ Shift` `P`        |
| 文字工具     | `T`                       | `T`                  |
| 容器工具     | `F`                       | `F`                  |
| 移动视图工具 | `H`                       | `H`                  |
| 取色器       | `⌃ Control` `C`/`I`       | `I`                  |
| 资源搜索器   | `M`                       | `M`                  |
| 评论工具     | `C`                       | `C`                  |
| 校对工具     | `E`                       | `E`                  |
| 圈画工具     | `D`                       | `D`                  |
